# 📱 OG To-Do List

A simple Android To-Do List app built using Kotlin.

## 🚀 Features

* Add tasks
* Update tasks
* Delete tasks
* Persistent storage using Room DB

## 🛠️ Tech Stack

* Kotlin
* Android Studio
* Room Database
* MVVM Architecture

## ▶️ How to Run

1. Clone the repo
2. Open in Android Studio
3. Run on emulator/device

## 📌 Future Improvements

* Add reminders
* Cloud sync
* Better UI
